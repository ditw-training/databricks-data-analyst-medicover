# Trainer Guide

Roboczy przewodnik dla szkolenia `Databricks-Data-Analyst-Medi`.

## Format

- Czas: 7 godzin.
- Uklad: 2 sesje po 3,5 godziny albo 1 dzien szkoleniowy.
- Jezyk: polski.
- Platforma: Databricks SQL Warehouse, Unity Catalog, notebooki Databricks.

## Przebieg

### Przed szkoleniem

1. Uruchomic `setup/00_pre_config.ipynb`.
2. Uruchomic `data/generate_training_dataset.ipynb`.
3. Zweryfikowac dostep uczestnikow do katalogu, schematow i SQL Warehouse.

### Sesja 1

1. Modul 1: SQL Warehouse + Notebooks.
2. Modul 2: Gold/KPI/best practices.
3. Warsztat 1: budowa analitycznego zestawu KPI w Gold.

### Sesja 2

1. Modul 3: dataset i semantyka pod Power BI.
2. Warsztat 2: dataset Power BI z nastawieniem na wydajnosc.
3. Modul 4: performance, automatyzacja i CI/CD na poziomie orientacji.

## Uwagi zakresowe

Pelny Jobs/Lakeflow/CI-CD hands-on nie miesci sie w tej wersji. Ten zakres
powinien byc traktowany jako wariant rozszerzony.
